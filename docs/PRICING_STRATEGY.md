## Vue d'ensemble

### Philosophie pricing

NEXUS propose une solution **tout-en-un** pour les professionnels (salons, restaurants, etc.) avec:
- **Agents IA omnicanal** (Téléphone, WhatsApp, Web)
- **Gestion métier complète** (Clients, RDV, Facturation, Stock, RH)
- **Marketing automatisé** (Posts IA, Images IA, SEO)
- **Intelligence business** (Analytics, Prédictions, SENTINEL)

### Positionnement tarifaire

| Segment | Plan | Prix | Cible |
|---------|------|------|-------|
| Entry | Starter | 99€/mois | Petits établissements, solo |
| Growth | Pro | 249€/mois | Établissements en croissance |
| Enterprise | Business | 499€/mois | Multi-établissements, franchises |

### Principe de modularité

Les clients peuvent **retirer des canaux** qu'ils n'utilisent pas:
- Sans Téléphone: réduction variable selon plan
- Sans WhatsApp: réduction variable selon plan
- Web only: prix le plus bas

## Optimisations techniques

### 1. Claude - Prompt Caching (CRITIQUE)

**Impact: -60% sur coûts Claude**

```javascript
// Les prompts système et contexte sont cachés
const cachedSystemPrompt = `
  Tu es l'assistant du salon {{nom_salon}}.
  Horaires: {{horaires}}
  Services: {{services}}
  // ... contexte métier (500+ tokens)
`;

// Seule la question client est facturée full price
// Cache read = 0.1x le prix normal
```

**Implémentation:**
- Activer `cache_control` dans les appels API Anthropic
- TTL de 5 minutes minimum pour conversations
- Réutiliser le contexte salon entre les messages

### 2. Routing intelligent des modèles

**Impact: -50% sur coûts chat**

```
┌─────────────────────────────────────────────────────────────┐
│                    ROUTING IA                               │
├─────────────────────────────────────────────────────────────┤
│  Question simple (FAQ, horaires, prix)                      │
│  └──→ Cache Redis (GRATUIT)                                 │
│                                                             │
│  Question standard (réservation, dispo)                     │
│  └──→ Claude Haiku (~0.001€)                                │
│                                                             │
│  Question complexe (réclamation, conseil)                   │
│  └──→ Claude Sonnet (~0.015€)                               │
│                                                             │
│  Génération contenu (posts, articles)                       │
│  └──→ Claude Batch API (-50%)                               │
└─────────────────────────────────────────────────────────────┘
```

**Critères de routing:**
- Longueur de la question
- Mots-clés détectés
- Historique de conversation
- Complexité estimée

### 3. OpenAI TTS au lieu d'ElevenLabs

**Impact: -90% sur coûts voix**

| Service | Coût/1K caractères | Qualité |
|---------|-------------------|---------|
| ElevenLabs | $0.24 | Premium |
| **OpenAI TTS** | **$0.015** | Très bon |

**Migration:**
```javascript
// Avant (ElevenLabs)
const speech = await elevenlabs.generate({
  text: response,
  voice: "voice_id"
});

// Après (OpenAI TTS)
const speech = await openai.audio.speech.create({
  model: "tts-1",
  voice: "nova", // ou "alloy", "echo", "fable", "onyx", "shimmer"
  input: response
});
```

### 5. Cache Redis pour FAQ

**Impact: -50% requêtes IA**

```javascript
const FAQ_CACHE = {
  "horaires": "Nous sommes ouverts du mardi au samedi, 9h-19h.",
  "prix_coupe_femme": "La coupe femme est à 45€.",
  "adresse": "123 rue de Paris, 75001 Paris",
  // ... réponses pré-calculées
};

async function handleMessage(question) {
  // 1. Chercher dans le cache FAQ
  const cached = await redis.get(`faq:${tenantId}:${hash(question)}`);
  if (cached && similarity(question, cached.question) > 0.9) {
    return cached.response; // GRATUIT
  }

  // 2. Sinon, appeler Claude
  const response = await claude.chat(question);

  // 3. Cacher si c'est une question fréquente
  if (isFrequentQuestion(question)) {
    await redis.set(`faq:${tenantId}:${hash(question)}`, {
      question, response
    }, { ex: 86400 }); // 24h
  }

  return response;
}
```

### 6. Batch API pour génération de contenu

**Impact: -50% sur posts IA**

```javascript
// Au lieu de générer les posts un par un
// On les batch et on utilise l'API asynchrone

const batchRequest = await anthropic.batches.create({
  requests: posts.map(post => ({
    custom_id: post.id,
    params: {
      model: "claude-3-haiku-20240307",
      messages: [{ role: "user", content: post.prompt }]
    }
  }))
});

// Récupérer les résultats plus tard (jusqu'à 24h)
const results = await anthropic.batches.results(batchRequest.id);
```
