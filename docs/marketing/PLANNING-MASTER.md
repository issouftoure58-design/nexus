### CHANTIER PRIORITAIRE — Ligne Demo IA

**Le concept** : Un numero de telephone NEXUS public. Le prospect appelle, l'IA repond et fait une demo live d'elle-meme. C'est la meilleure preuve de valeur possible — le prospect teste le produit AVANT de s'inscrire.

**Ce que l'IA fait sur la ligne** :
- Se presente : "Bonjour, je suis l'assistante IA NEXUS. Je suis exactement le type d'IA que vous auriez pour votre business."
- Repond aux questions : comment ca marche, combien d'appels simultanes, adaptation metier, personnalisation
- Explique l'installation : "Creez votre compte sur nexus-ai-saas.com, parametrez votre activite, demandez l'activation — on configure tout en 48h"
- Donne les tarifs : "Gratuit pour commencer, puis 29€/mois avec 1000 credits IA inclus"
- Pousse vers la creation de compte

**Hook marketing** : `"Appelez ce numero. Une IA va vous repondre. Jugez par vous-meme."`

**Mise en place** :
| Etape | Action | Delai |
|-------|--------|-------|
| 1 | Acheter numero OVH (06/07 ou 01/09) | 1 jour |
| 2 | Creer prompt `demoAgentPrompt.js` | 1 jour |
| 3 | Configurer SIP trunk OVH → backend | 1 jour |
| 4 | Tester le flow complet | 1 jour |
| 5 | Ajouter le numero sur landing, flyers, captions | 1h |
| 6 | Poster sur groupes FB + reseaux | continu |

**Impact sur toute la comm** :
- Chaque post FB finit par "Appelez le 0X XX XX XX XX — l'IA vous repond"
- Chaque flyer a le numero en gros + QR code
- Chaque caption reseaux = "Vous ne nous croyez pas ? Appelez."
- Le terrain devient 10x plus facile : "Tenez, appelez ce numero maintenant, vous allez voir"

**Priorite** : AVANT la tournee terrain (semaine 17). Idealement pret semaine 16.
