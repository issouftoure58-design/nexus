## REGLES DE PUBLICATION

- **Maximum 1 post par jour par plateforme** (sauf stories)
- **Stories** : 2-3 par jour max (teasing, behind-the-scenes, sondages)
- **Espacement** : minimum 24h entre 2 posts sur la meme plateforme
- **Heures optimales** :
  - Instagram / Facebook : **12h00 - 14h00** (pause dejeuner)
  - TikTok : **18h00 - 21h00** (fin de journee)
  - LinkedIn : **8h00 - 10h00** (matin pro)
- **Cross-posting** : poster le meme contenu sur IG + FB en meme temps = OK
- **TikTok** : contenu video uniquement (pas de carrousels photo)

## COMMENT PUBLIER (etape par etape)

1. Regarde le calendrier → trouve la date du jour
2. Ouvre le dossier correspondant dans `stock-pret-a-publier/`
3. Transfère les images/video sur ton telephone (AirDrop)
4. Ouvre Instagram / Facebook / TikTok
5. Cree le post (Reel pour video, Publication pour image/carrousel)
6. Ouvre `caption.txt` → copie-colle le texte
7. Ajoute localisation : France
8. Pour IG : active "Partager aussi sur Facebook"
9. Publie
10. Mets a jour le SUIVI ci-dessus → change "A POSTER" en "POSTE ✅"
