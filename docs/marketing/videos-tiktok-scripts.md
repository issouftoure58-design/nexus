## CONSEILS DREAMINA

1. **Générer chaque scène séparément** puis assembler dans CapCut
2. **Garder le même personnage** — régénérer jusqu'à avoir un visage cohérent
3. **Overlay texte dans CapCut** (pas dans Dreamina) pour contrôler le timing
4. **Format 9:16** obligatoire (TikTok/Reels)
5. **Durée totale : 15-30s max** — TikTok favorise les vidéos courtes
6. **Musique trending** — chercher sur TikTok "motivational transformation" ou "glow up business"
